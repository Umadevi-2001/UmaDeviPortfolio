# Personal Portfolio Website

This is a **Personal Portfolio Website** developed using **HTML and CSS** to showcase my skills, services, and contact information.  
The website highlights my projects, technical skills, and provides ways to connect with me.

## Features

- Responsive navigation bar
- Home section with introduction
- About section describing my background
- Services section showcasing my skills
- Skills section with progress indicators
- Contact form for communication
- Social media links (LinkedIn, GitHub, Instagram)
- Clean and modern UI design

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
- Responsive Layout (Flexbox & Grid)

## Project Structure

```
portfolio-website
│
├── index.html
├── style-3.css
├── image.jpeg
└── README.md
```

## Sections in the Website

1. **Home** – Introduction and social media links  
2. **About** – Information about my background and career goals  
3. **Services** – Web development, database development, and Python development  
4. **Skills** – Technical skills with progress bars  
5. **Contact** – Contact form for communication  

## How to Run the Project

1. Download or clone the repository

```
git clone https://github.com/umadevi-2001/portfolio-website.git
```

2. Open the project folder

3. Open `index.html` in any web browser
